Configuration Files

Trindex supports configuration via YAML files in standard locations (following XDG Base Directory Specification).

Config File Locations (in order of precedence)

1 Explicit path (highest priority)
  • --config /path/to/config.yaml flag
  • TRINDEX_CONFIG environment variable
2 Current directory
  • ./trindex.yaml
  • ./.trindex.yaml
3 User config directory (XDG)
  • ~/.config/trindex/config.yaml Linux
  • ~/Library/Application Support/trindex/config.yaml macOS
  • %APPDATA%\trindex\config.yaml Windows
4 Legacy locations
  • ~/.trindex.yaml
  • ~/.trindex/config.yaml
5 System-wide (Unix only)
  • /etc/trindex/config.yaml
  • /etc/trindex.yaml

Quick Setup

Create your config file:

# Linux/macOS
mkdir -p ~/.config/trindex
cat > ~/.config/trindex/config.yaml << 'EOF'
database_url: "postgres://trindex:trindex@localhost:5432/trindex?sslmode=disable"
embed_base_url: "http://localhost:11434/v1"
embed_model: "nomic-embed-text"
embed_api_key: "ollama"
embed_dimensions: 768
EOF

# Verify it works
trindex doctor

Configuration Options

Database Settings

Option Default Description
database_url postgres://trindex:trindex@localhost:5432/trindex?sslmode=disable PostgreSQL connection string
db_max_conns 100 Maximum database connections
db_min_conns 10 Minimum database connections
db_max_conn_lifetime_minutes 60 Maximum connection lifetime (minutes)
db_max_conn_idle_time_minutes 30 Maximum idle time before closing (minutes)

Embedding Settings

Option Default Description
embed_base_url http://localhost:11434/v1 Embedding API base URL (Ollama default)
embed_model nomic-embed-text Embedding model name
embed_api_key ollama API key for embedding service
embed_dimensions 768 Vector dimensions (must match model)
embed_max_retries 3 Maximum retry attempts
embed_retry_delay_ms 1000 Delay between retries (milliseconds)
embed_request_timeout_sec 30 Request timeout (seconds)

HTTP Server Settings

Option Default Description
http_host 0.0.0.0 HTTP server bind address
http_port 9636 HTTP server port
http_api_key (empty) API key for authentication (optional)

Search Settings

Option Default Description
default_namespace default Default namespace for memories
default_top_k 10 Default number of search results
default_similarity_threshold 0.7 Minimum similarity score (0.0-1.0)
hybrid_vector_weight 0.7 Weight for vector search (0.0-1.0)
hybrid_fts_weight 0.3 Weight for full-text search (0.0-1.0)

HNSW Index Settings

Option Default Description
hnsw_m 16 Number of connections per layer (higher = more accurate, more memory)
hnsw_ef_construction 64 Candidate list size during index building
hnsw_ef_search 40 Candidate list size during search

Example Configuration

# Trindex Configuration File
# Place at: ~/.config/trindex/config.yaml

# Database connection
database_url: "postgres://trindex:trindex@localhost:5432/trindex?sslmode=disable"

# Embedding configuration (Ollama)
embed_base_url: "http://localhost:11434/v1"
embed_model: "nomic-embed-text"
embed_api_key: "ollama"
embed_dimensions: 768

# For OpenAI instead:
# embed_base_url: "https://api.openai.com/v1"
# embed_model: "text-embedding-3-small"
# embed_api_key: "sk-your-key-here"
# embed_dimensions: 1536

# HTTP Server settings
http_host: "0.0.0.0"
http_port: "9636"
http_api_key: ""  # Set to enable API authentication

# Default search settings
default_namespace: "default"
default_top_k: 10
default_similarity_threshold: 0.7

# Hybrid search weights (should sum to 1.0)
hybrid_vector_weight: 0.7
hybrid_fts_weight: 0.3

# HNSW index tuning
hnsw_m: 16
hnsw_ef_construction: 64
hnsw_ef_search: 40

# Connection pooling
db_max_conns: 100
db_min_conns: 10
db_max_conn_lifetime_minutes: 60
db_max_conn_idle_time_minutes: 30

# Retry configuration
embed_max_retries: 3
embed_retry_delay_ms: 1000
embed_request_timeout_sec: 30

Configuration Precedence

Configuration values are loaded in this order (later overrides earlier):

  1. Default values - Built-in defaults
  2. Config file - From standard locations or --config flag
  3. Environment variables - Override config file
  4. Command-line flags - Highest priority

Tip: Use config files for defaults and environment variables for deployment-specific overrides. This makes it easy to have different settings for development, staging, and production.

Environment Variables

All configuration options can be set via environment variables using UPPER_CASE with underscores:

export DATABASE_URL="postgres://user:pass@host/db"
export EMBED_MODEL="nomic-embed-text"
export DEFAULT_TOP_K="20"
export HTTP_PORT="3000"

# Or run directly
DATABASE_URL="postgres://..." EMBED_MODEL="..." trindex server
